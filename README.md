# Android SpreadBar example
## Description
This application is demo for "spread bar" widget

## Requirements

1. Install the following software:

  [Java] [java]

  [Android SDK] [android sdk]

  [Gradle] [gradle]

2. Run commannd
	`gradlew :app:clean :app:assembleDebug`

3. Install on device:
	`gradlew :app:installDebug`
